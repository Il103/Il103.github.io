# BERU • Il103 GitHub Pages Site

Static multi-page portfolio built for direct upload to GitHub Pages.

## Repository name
For the main user site, create the repository with this exact name:

`Il103.github.io`

Then put all files from this folder in the root of the repository.

- Current email: `beru.spider.team@gmail.com`

## Edit later from one file
Open:

`assets/js/config.js`

You can change:
- owner name
- tagline
- email
- social links
- stack
- services
- team members
- terminal text

## Publish
1. Create the repo `Il103.github.io`
2. Upload these files so `index.html` is in the root
3. Go to **Settings → Pages**
4. Set **Source** to **Deploy from a branch**
5. Choose **main** and **/ (root)**
6. Save

## Optional custom domain later
Keep the repository as:

`Il103.github.io`

Then when you buy a domain, add it in **Settings → Pages → Custom domain**.

For example:
- `beru.dev`
- `www.beru.dev`

The repo name does not need to change for a custom domain.
